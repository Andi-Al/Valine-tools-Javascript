# Chatbot Script Generator

> **Category**: Developer Tools
> **Icon**: 🤖
> **Type**: Generator

## 📝 Description
**How it works:** Generate a fully functional chatbot widget with conversation flows (trigger → response), typing animation, quick reply buttons, and customizable theme. Outputs embeddable HTML/CSS/JS code that can be added to any website.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Set bot name and greeting message
3. Add conversation flows (trigger keywords → bot response)
4. Set fallback response for unmatched inputs
5. Configure widget position, theme color, input placeholder
6. Enable/disable typing animation
7. Select language (English / Bahasa Indonesia)
8. Click "Generate Chatbot"
9. Preview the interactive chatbot in real-time
10. Copy embed code or download as .html file

## 📁 File Structure
```
chatbot script generator/

```

## ⚡ Features
- Dynamic conversation flow builder (trigger → response)
- Comma-separated trigger keywords
- Fallback response for unmatched inputs
- Typing animation indicator
- Customizable widget position (bottom-right / bottom-left)
- Theme color picker
- Custom input placeholder text
- Language selection (English / Bahasa Indonesia)
- Live interactive preview (testable chatbot)
- 3 output tabs: Embed Code, HTML, Full Page
- Copy code and download as .html
- Auto-generate on input change (debounced)
- Mobile responsive design

## 🛠️ Technical Details
- **HTML5** - Semantic markup
- **CSS3** - Responsive design + widget animations
- **JavaScript (ES6+)** - No external dependencies
- **Standalone** - All assets self-contained

If you encounter issues:
