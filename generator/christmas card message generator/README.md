# Christmas Card Message Generator

> **Category**: Message Generator
> **Icon**: 🎄
> **Type**: Generator

## 📝 Description
**How it works:** Generate personalized Christmas card messages based on recipient category (family, friends, coworkers, clients, teachers, romantic) and tone (warm, funny, religious, professional, heartfelt). Combines greeting, personal sentiment, wish, and sign-off patterns with optional Christmas quotes.

## 🚀 How to Use
1. Open `index.html` in your browser
2. Enter your name (sender) and recipient name
3. Select category (Family, Friends, Coworkers, etc.)
4. Select tone (Warm, Funny, Religious, Professional, Heartfelt)
5. Check "Include Christmas Quote" for added quotes
6. Add an optional personal message
7. Set number of messages to generate (1-10)
8. Click "Generate Messages"
9. View generated messages in card-style preview
10. Copy individual messages or copy all at once

## ⚡ Features
- 6 categories: Family, Friends, Coworkers, Clients, Teachers, Romantic
- 5 tones: Warm, Funny, Religious, Professional, Heartfelt
- 30 unique message templates (6 × 5 combinations, each with 3 variations)
- 10 Christmas quotes from famous authors and Bible verses
- Personalization with sender and recipient names
- Optional personal message (P.S.)
- Generate 1-10 unique messages at once
- Card-style preview with copy per message
- Copy all messages at once
- Regenerate for new variations
- Auto-generate on input change (debounced)
- Christmas-themed UI (red & green gradient)
- Mobile responsive design
